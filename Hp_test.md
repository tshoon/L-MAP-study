# 하이퍼 파라미터 튜닝

## 평가 기준
### F1-score
- 모델의 정밀도(Precision)와 재현율(Recall)의 조화 평균


### PDSD(Prediction Distribution Shift Distance)
- 데이터 분포가 변할 때 모델이 얼마나 민감하게 반응하는지를 평가
- 값이 높을수록: 모델 예측이 데이터 분포 변화에 더 민감함
